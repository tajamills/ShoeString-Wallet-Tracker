## What We Do

Crypto Bag Tracker is a web-based software platform that helps cryptocurrency holders track their digital asset portfolios and prepare tax documentation required by the IRS.

### The Problem We Solve

Cryptocurrency investors face significant challenges with tax compliance:

1. **Fragmented Holdings** - Users hold assets across multiple wallets and exchanges
2. **Complex Tax Rules** - IRS requires tracking cost basis for every transaction
3. **Manual Process** - Without software, users must manually calculate gains/losses
4. **Audit Risk** - Incorrect reporting can result in penalties and interest

### Our Solution

Our platform automatically:
- Connects to 12 different blockchains to read wallet balances
- Imports transaction history from 9+ cryptocurrency exchanges
- Calculates cost basis using IRS-accepted FIFO method
- Generates Form 8949 and Schedule D reports for tax filing
- Traces asset origins for compliance and audit purposes

## Technology

### Platform Architecture
- **Frontend:** React web application
- **Backend:** Python/FastAPI REST API
- **Database:** MongoDB (cloud-hosted)
- **Hosting:** Render (cloud platform)
- **Payments:** Stripe

### Security Measures
- All data transmitted over HTTPS/TLS
- Passwords hashed with bcrypt
- API keys encrypted at rest (AES-128)
- Read-only access to exchanges (cannot move funds)
- No storage of private keys or seed phrases

### Data Handling
- User data stored in encrypted database
- No sharing of user data with third parties
- Users can delete their account and data on request
- CCPA/GDPR compliant data practices
