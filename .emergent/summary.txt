<product_requirements>
The product is Crypto Bag Tracker, a web application for analyzing cryptocurrency wallet transactions, primarily for tax reporting. The app is deployed on Render and uses a tiered subscription model monetized via Stripe.

**Tiers:**
*   **Free:** Ethereum-only analysis.
*   **Premium:** Unlimited analyses, multi-chain support, CSV export, advanced analytics.
*   **Pro:** All Premium features, plus Analyze All Chains aggregation, Bitcoin xPub support, and custom chain requests.

**Core Features Implemented:**
*   **Multi-Chain Analysis:** Supports Ethereum, Bitcoin, Polygon, Arbitrum, BSC, and Solana.
*   **Stripe Subscriptions:** Manages recurring monthly payments for Premium and Pro tiers, including upgrades, downgrades, and coupon support.
*   **Saved Wallets:** Allows authenticated users to save and manage wallet addresses.
*   **Advanced Export:** Paginated CSV export for complete transaction history.
*   **HD Wallet Support:** Analyzes entire Bitcoin wallets using xPub keys (Pro feature).
*   **Data Clarity:** Displays current balance, lifetime totals, and a running balance in transaction history to avoid confusion.
*   **Custom Branding:** App name, logo, and favicon updated to Crypto Bag Tracker.

The current development focus is on building a comprehensive tax calculation engine.
</product_requirements>

- **Deployment:** Render for both frontend and backend services.
- **Pricing Data:** CoinGecko API for real-time and historical crypto prices.
